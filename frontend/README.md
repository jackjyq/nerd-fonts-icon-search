# Nerd Fonts Icon Search

![](https://img.shields.io/badge/nodejs-20.10%20LTS-green)

## Quick Start

```shell
# in `$PROJECT_ROOT/frontend` folder
npm install

npm start
```

## Configuration

Copy `.env` to `.env.local` and edit `.env.local`
